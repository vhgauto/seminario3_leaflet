# Mapas Interactivos

Repositorio correspondiente a la sección [Interactive maps](https://vhgauto.github.io/seminario3-gulich/#/sec-interactive) del [seminario **Better maps with R**](https://vhgauto.github.io/seminario3-gulich/) a cargo del MS Víctor Gauto.

Contiene mapas interactivos creados en `R` con los siguientes paquetes:

- [ggiraph](https://davidgohel.github.io/ggiraph/)
- [terra](https://rspatial.org/)
- [leaflet](https://rstudio.github.io/leaflet/)
- [tmap](https://github.com/r-tmap/tmap)
- [mapview](https://github.com/r-spatial/mapview)
